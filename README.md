# README

## Getting started

Install hugo:

```bash
brew install hugo
```

Or go to <https://github.com/gohugoio/hugo/releases>.

Start the server:

```bash
hugo server
```

Add a post with

```bash
hugo new posts/my-post.md
```

To add a new page in the main menu, create the page in the `content/` directory,
and add it to `config.toml`.
